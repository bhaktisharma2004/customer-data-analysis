Data analytics project showing customer behaviour analysis 
This project demonstrates an end-to-end data analytics workflow, starting from raw data ingestion to insight generation and visualization. The goal is to showcase practical skills in Python, SQL, Power BI, and reporting used in real-world analytics roles.

The project includes:

Loading and preprocessing a dataset using Python

Performing data cleaning and EDA

Running SQL queries on PostgreSQL/MySQL/SQL Server

Building an interactive Power BI dashboard

Creating a PPT report

Generating insights using Gamma (AI-assisted analysis)
